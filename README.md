# 🚀 Microsoft AI LiveStream Series PPT

A curated collection of PowerPoint presentations covering modern AI engineering concepts, Large Language Models (LLMs), Retrieval-Augmented Generation (RAG), AI Agents, Vector Embeddings, Vision Models, MCP (Model Context Protocol), Structured Outputs, Tool Calling, and AI Safety.

This repository centralizes educational slide decks used for AI livestreams, workshops, demos, and technical learning sessions.
